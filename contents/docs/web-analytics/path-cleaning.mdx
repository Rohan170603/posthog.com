## Managing rules with MCP

You can also manage path cleaning rules programmatically using the [PostHog MCP server](/docs/model-context-protocol). The `path-cleaning-rules-update` tool lets you append, insert, replace, remove, and reorder rules from your code editor or AI agent — with a built-in preview step so you can verify changes before saving. This is especially useful when you have many rules to manage or want to automate rule setup.
